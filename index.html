about

Kia ora, I'm Taiao, a full stack dev with 2.5 years of experience and based in Wellington NZ.

This portfolio site is a rewrite of a bloated React based site I built after my studies, with youthful optimism, in 2023. I've stripped it down to a single HTML file with some inline javascript and CSS to keep maintenance at a minimum for myself.
